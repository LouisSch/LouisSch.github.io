Monte Carlo Simulation : application to option pricing under Black-Scholes Model¶

Monte Carlo Principle¶

The problem to solve here, is the computation of the quantity $\mathbb{E}(g(X))$, where $X$ is some random variable and $g$ some function with properties so that the expectation is well-defined.
Indeed, when one wants to compute the price of a call option, it needs to compute $\mathbb{E}^*\left[e^{-rT}(S_T - K)^+\right]$, under the risk-neutral probability measure $\mathbb{P}^*$: it is a particular case of the problem presented above.

So we need an estimator for the expectation. We can use the Law of Large Numbers:
Let $(X_i)_{1\leq i\leq N}$ be a sequence of integrable, independently and identically distributed random variables. We have the following convergence: $$ \bar{X} = \frac{1}{N}\sum_{i=1}^N X_i \xrightarrow[N\to +\infty]{} \mathbb{E}(X_1) $$ Obviously, we also have: $$ \frac{1}{N}\sum_{i=1}^N g(X_i) \xrightarrow[N\to +\infty]{} \mathbb{E}(g(X_1)) $$ So if we know how to simulate $(X_i)_{1\leq i\leq N}$, we know how to compute the expectation.

Black-Scholes diffusion is: $$ dS_t=\mu S_tdt + \sigma S_tdB_t $$ where $(B_t)_{t\geq0}$ is a Brownian motion under $\mathbb{P}$.

With Girsanov's Theorem, we can find $\mathbb{P}^* \sim \mathbb{P}$ such that, under this new measure, we have: $$ dS_t = rS_tdt + \sigma S_t dW_t $$ where $(W_t)_{t\geq0}$ is a Brownian motion under $\mathbb{P}^*$, and $r$ is the risk-free rate.

We can fin the solution under $\mathbb{P}^*$: $$ S_t = S_0e^{(r- \frac{\sigma^2}{2})t + \sigma W_t} $$ , where $W_t \sim \mathcal{N}(0,t)$.

We need to simulate a Gaussian real-valued random variable.

1) Box-Müller formula¶

With $U_1$ and $U_2$ being independent and identically distributed. We have $U_1 \sim \mathcal{U}(0,1)$, the uniform law.
Then: $$ X_1 = \sqrt{-2log(U_1)}cos(2\pi U_2),\\ X_2 = \sqrt{-2log(U_1)}sin(2\pi U_2) $$ where $X_1, X_2$ are both distributed with $\mathcal{N}(0,1)$ and independent. With the central limit theorem: $$ \sqrt{t}X_1 \sim \mathcal{N(0,t)}\\ \sqrt{t}X_2 \sim \mathcal{N(0,t)} $$

# Simulation with Box-Müller
# Given two vectors of uniformely distributed (0,1) values

def SimulateGaussians(U1, U2, variance=1):
    result = []
    if len(U1) != len(U2):
        raise Exception("We need same length for U1 and U2.")
    for i in range(len(U1)):
        result.append([np.sqrt(-2*variance*np.log(U1[i]))*np.cos(2*np.pi*U2[i]), np.sqrt(-2*variance*np.log(U1[i]))*np.sin(2*np.pi*U2[i])])

    return result

3) Now we simulate our paths¶

We can write with a discretization scheme : $$ S(t+\Delta t) = S_te^{(r-\frac{\sigma^2}{2})\Delta t + \sigma \sqrt{\Delta t}Z},\\ Z\sim \mathcal{N}(0,1) $$

II) Price of a call option¶

We have the following payoff: $(S_T - K)^+$, so we want to compute $\mathbb{E}^*[e^{-rT}(S_T - K)^+]$.
We write: $\psi(x)=e^{-rT}(x - K)^+$, the problem is then: $\mathbb{E}^*[\psi(S_T)]$.
So we are going to apply this function to each path and compute the empirical estimator of $\mathbb{E}$.

II) Reduction of variance¶

Since the error of the Monte-Carlo is linked to the variance of the empirical estimator, we need to reduce that variance to get a more accurate result. We can also see that by building the confidence interval of the estimator, where the variance controls the size of the interval.

Indeed, if we write $\bar{X}_N = \frac{1}{N}\sum_{i=1}^NX_i$ (bias), then $\mathbb{E}(X) \in [\bar{X}_N -\alpha \frac{\sigma_N}{\sqrt{N}}, \bar{X}_N + \alpha \frac{\sigma_N}{\sqrt{N}}]$ where $\alpha$ is a quantile of $\mathcal{N}(0,1)$: if we take a $95\%$ confidence interval, then $\alpha = 1.96$. Hence to refine our estimate, we need to decrease $\sigma_N$ (since the rate of converge is $\frac{1}{\sqrt{N}}$, pretty slow, it is better to decrease $\sigma_N$).

1) Control variates¶

We can write: $$ \begin{align} \mathbb{E}(g(X)) = \mathbb{E}(g(X)) - \mathbb{E}(h(X)) + \mathbb{E}(h(X)) \end{align} $$ where $\mathbb{E}(h(X))$ can be computed easily.
We know that the Put payoff, $(K - S_T)^+$ is bounded, then it is easier to compute its price.

By the Call-Put parity: $$ C_0 - P_0 = S_0 - Ke^{-rT} \implies C_0 = \mathbb{E}^*(e^{-rT}(K-S_T)^+) + S_0 - Ke^{-rT}. $$ It is in the form of equation $(1)$.

2) Antithetic variables¶

If we have $X$ a real-values random variable, and $\Gamma$ a transformation of $\mathbb{R}$ such that the law of $\Gamma(X)$ is the same as the law of $X$, we can write: $$ \mathbb{E}(g(X)) = \frac{1}{2}\Bigl[g(X) + g(\Gamma(X))\Bigr] $$

For the standard Monte-Carlo ($(X_i)_{i\in I}$ are identically distributed and independent): $$ \frac{1}{2n}\sum_{i=1}^{2n}g(X_i) = \frac{1}{n}\sum_{i=1}^{2n}\frac{1}{2}g(X_i) = I_{2n}. $$ Then: $$ Var(I_{2n}) = \frac{1}{2n}Var(g(X_1)) $$

If we compute the variance of the new estimator: $$ \begin{align} Var\Bigl[\frac{1}{n} \sum_{i=1}^{2n} \frac{1}{2} \bigl(g(X_i) + g(\Gamma(X_i))\bigr)\Bigr] & =\frac{1}{4n}\Bigl[Var(g(X_1)) + Var\bigl(g(\Gamma(X_1))\bigr) + 2Cov\Bigl(g(X_1), g(\Gamma(X_1))\Bigr)\Bigr]\\ & = \frac{1}{2n}\Bigl( Var\bigl(g(X_1)\bigr) + Cov\Bigl(g(X_1), g(\Gamma(X_1))\Bigr) \Bigr) \end{align} $$

We can deduce that if $Cov\Bigl(g(X_1), g(\Gamma(X_1))\Bigr) < 0$, the variance of the estimator is descreased with this formula.
For our pricing, $\forall \space 0 < t \leq T, W_t \sim \mathcal{N}(0,t)$.
We can take $\Gamma(W_t) = -W_t$, as $W_t$ and $-W_t$ have the same law, and $Cov\Bigl(W_t, -W_t\Bigr) = -t < 0$.

    return np.exp(-r*T)*max(x - K, 0)

def antithetic_mean(prices, prices_antithetic):
    return (1/(2*len(prices)))*(sum(prices) + sum(prices_antithetic))
